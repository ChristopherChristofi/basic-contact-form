# simple-contact-form

## Description:

Simple contact form that utilises formsubmit.co methods to an email account.

## Features:

- Simple form submit
- Text input
- Basic page routing

## Build Instructions:

1. **Install dependencies**
```sh
npm install
```

2. **Quick Start:**
```sh
npm start
```

3. **Development start:**
```sh
npm run start:dev
```
